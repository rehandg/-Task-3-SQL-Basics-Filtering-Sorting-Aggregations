## Steps Performed

1. Installed and verified MySQL Server.
2. Created a new database named `chinook`.
3. Imported the Chinook MySQL dataset using the command-line interface.
4. Verified successful import by listing available tables.
5. Explored the dataset using basic `SELECT` queries.

8. Used aggregate functions such as `SUM`, `AVG`, and `COUNT`.

12. Saved all SQL queries in a `queries_task3.sql` file.
